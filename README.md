# Ecommerce-Project
Analyzed customer data for an online clothing retailer to help prioritize between mobile app and website platforms. Explored key features like Avg. Session Length, Time on App, Time on Website, Length of Membership, and Yearly Amount Spent. Built predictive models to identify factors contributing to annual income and provided actionable insights to enhance customer engagement and sales strategies. Evaluated model performance using metrics such as R-squared and RMSE.
